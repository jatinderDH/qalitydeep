# QAlityDeep Eval API Playground

Resizable 3-panel UI for the eval API. Drag the vertical dividers to resize.

## Run

1. Start the API: `python -m qalitydeep.cli serve-api --port 8000`
2. From this folder: `npm install && npm run dev`
3. Open http://localhost:5173

API URL is configurable in the middle panel (default `http://localhost:8000`). Set `VITE_API_URL` in `.env` to change the default.
